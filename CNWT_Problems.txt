"Internet seems fine but every website says it can't be reached unless I use the IP address directly."

🧪 3. DNS Timeout (tricky)

"Web pages take forever to load and sometimes eventually open, but ping to 8.8.8.8 works instantly."

🧪 4. IP Conflict (real-world style)

"My laptop keeps disconnecting randomly and sometimes shows 'another device is using this IP address' warning."

🧪 5. DHCP Failure (edge case)

"My system shows limited connectivity and the IP starts with 169 even after reconnecting multiple times."

🧪 6. Gateway Unreachable (confusable with router issue)

"I checked my IP settings and everything looks fine but I cannot ping my gateway at all."

🧪 7. Network Adapter Issue (hardware-like)

"My ethernet cable is plugged in but my computer says no network devices found."

🧪 8. Subnet Mismatch (advanced case)

"I manually set an IP but I still can't reach the gateway even though both are in the same range."

🧪 9. Hybrid Case (hard for model)

"WiFi connects, I get an IP, I can ping the router, but websites are inconsistent and sometimes fail to load."
